Mini-Projet HTML

Ce mini-projet, conçu comme une introduction au projet collectif, a pour objectif de fournir des informations sur le contenu extrait d'une série d'URLs.

Un tableau est mis à votre disposition pour visualiser les données collectées. Il contient les éléments suivants :

  • L'URL concernée
  • Le code HTTP de la requête
  • L'encodage utilisé
  • Le nombre de mots détectés
